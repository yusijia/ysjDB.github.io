Problem Description

把一个偶数拆成两个不同素数的和，有几种拆法呢？

Input

输入包含一些正的偶数，其值不会超过10000，个数不会超过500，若遇0，则结束。

Output

对应每个偶数，输出其拆成 不同 素数的个数，每个结果占一行。

Sample Input

30
26
0

Sample Output

3
2

分析：

这对素数的范围是[2, date / 2]
