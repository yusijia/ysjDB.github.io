range

常用范围

  • 32位机的

  • 注意：有符号的第一位是符号位

  • unsigned int 0～4294967295 (1 << 32) - 1 (32位)

  • int -2147483648～2147483647 (1 << 31) - 1 (31位 + 1位)

  • long long的最大值：9223372036854775807 (1LL << 63) - 1 (63位 + 1位)

  • long long的最小值：-9223372036854775808

  • unsigned long long的最大值：18446744073709551615 （1ULL << 64） - 1 (64位)

  • 对应输出：

  • long long %I64d %lld

  • unsigned long long %I64u %llu
