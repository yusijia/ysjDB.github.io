费马小定理

费马小定理：

  • a的p-1次方除以p的余数等价于1

  • =>a^(p - 1) ≡ 1 (mod p)

  • 推出：a的p-2次方除以p的余数等价于1/a

  • -> a^(p-2) % p ≡ 1/a （1/a已经在[0, p-1]的范围之间了）

  • -> a^(p-2) % p ≡ (1/a) % p （再对1/a取一次模也不变）
    这就可以看成同余定理除法方面的应用：

(1 / a) % p ≡ a^(p - 2) % p

同余定理

同余定理：

  1. 加法： (a+b)%m=(a%m+b%m)%m
  2. 减法： (a-b)%m=(a%m-b%m+m)%m
  3. 乘法： (ab)%m=(a%mb%m)%m
  4. 除法 用费马小定理转化为乘法做
  5. mod n使数据范围变为0~n-1
